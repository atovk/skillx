# 练习题目录

通过实践练习来巩固你的 Claude Code Skills 开发技能。

## 练习级别

### 初级练习 (beginner/)

适合初学者的基础练习：

1. **[练习 1: 创建你的第一个 Hello Skill](beginner/exercise-01-hello-skill.md)**
   - 学习 Skill 的基本结构
   - 创建简单的问候 Skill

2. **[练习 2: 使用工具读取文件](beginner/exercise-02-simple-tool.md)**
   - 学习如何使用工具
   - 理解 allowed-tools

3. **[练习 3: 完善元数据](beginner/exercise-03-adding-metadata.md)**
   - 学习编写高质量的 description
   - 了解元数据字段

4. **[练习 4: 实现基础工作流](beginner/exercise-04-basic-workflow.md)**
   - 设计多步骤任务
   - 组织清晰的工作流

5. **[练习 5: 测试你的 Skill](beginner/exercise-05-testing-basics.md)**
   - 学习测试方法
   - 调试和修复问题

### 中级练习 (intermediate/)

适合有经验的学习者：

1. **练习 6: 实现渐进式披露** - 组织复杂内容
2. **练习 7: 使用工具限制** - 限制可用工具
3. **练习 8: 创建条件工作流** - 根据条件执行不同逻辑
4. **练习 9: 构建模板系统** - 使用模板提高复用性
5. **练习 10: 添加错误处理** - 健壮的错误处理

### 高级练习 (advanced/)

适合想要深入的学习者：

1. **练习 11: 实现协调器模式** - 复杂的多步骤工作流
2. **练习 12: 创建反馈循环** - 根据结果调整行为
3. **练习 13: 集成 MCP 工具** - 使用外部 MCP 工具
4. **练习 14: 构建多文件 Skill** - 组织大型 Skill
5. **练习 15: 发布和分享** - 发布你的 Skills

## 如何使用练习

### 1. 按顺序完成

建议从初级练习开始，按顺序完成：

```
beginner/ → intermediate/ → advanced/
```

### 2. 每个练习的步骤

1. **阅读任务描述**
   - 理解练习要求
   - 查看学习目标

2. **查看提示**
   - 如果卡住了，查看提示部分
   - 提示提供了有用的指导

3. **动手实现**
   - 创建你的 Skill
   - 测试功能

4. **对照验收标准**
   - 检查是否满足所有要求
   - 自我评估完成度

5. **查看参考答案**
   - 完成后查看参考答案
   - 对比和学习不同方法

6. **尝试延伸挑战**
   - 完成基本要求后，尝试挑战
   - 深化理解和技能

### 3. 获取帮助

如果遇到困难：

- 查看相关文档
- 参考示例 Skills
- 查看参考答案
- 提问社区

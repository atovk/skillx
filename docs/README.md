## 学习路径推荐

### 路径 A：快速上手（1 小时）

适合想要快速开始的学习者：

1. [快速入门](00-getting-started.md) - 10 分钟
2. [创建第一个 Skill](tutorials/tutorial-01-first-skill.md) - 20 分钟
3. [结构剖析](02-skill-anatomy.md) - 15 分钟
4. [最佳实践](03-best-practices.md) - 15 分钟

**结果**：能够创建简单的 Skills

### 路径 B：系统学习（1 天）

适合想要全面理解的学习者：

1. 所有核心文档（5 个）- 1.5 小时
2. 所有教程（5 个）- 2 小时
3. 完成初级练习 - 3 小时

**结果**：扎实的 Skills 开发基础

### 路径 C：深入学习（1 周）

适合想要精通的学习者：

1. 所有文档 - 4 小时
2. 所有教程 - 3 小时
3. 所有练习 - 20 小时
4. 研究示例 Skills - 10 小时

**结果**：能够开发生产级 Skills
